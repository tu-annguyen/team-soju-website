-# Server Components
- The main application logic is in apps/api-server/src.
- Feature flags and configuration settings are in apps/api-server/src/config.
- All API request handlers are in apps/api-server/src/routes.

# Frontend Code Structure
- The UI components for the website are in apps/web-app/src/components.
- Styles for the AI interface are in apps/web-app/src/styles.

# Data & Storage
- Data Model definitions are stored in apps/api-server/src/models.

# Test Location
- Tests belong in apps/web-app/test for UI testing and apps/api-server/test for server testing
