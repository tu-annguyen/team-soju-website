-# Server Components
- The main application logic is in server/src.
- Feature flags and configuration settings are in server/src/config.
- All API request handlers are in server/src/routes.

# Frontend Code Structure
- The UI components for the website are in client/src/components.
- Styles for the AI interface are in client/src/styles.

# Data & Storage
- Data Model definitions are stored in server/src/models.

# Test Location
- Tests belong in /client/test for UI testing an /server/test for server testing
