## ðŸ§ž NPM Commands

All commands are run from the root of the project:

| Command                   | Action                                                                |
| :------------------------ | :-------------------------------------------------------------------- |
| `npm install`             | Installs dependencies                                                 |
| `npm run dev:web`         | Starts local dev server at `localhost:4321` for the frontend web app  |
| `npm run dev:api`         | Starts local dev server at `localhost:3001` for the backend API       |
| `npm run dev`             | Concurrently starts local dev servers for the frontend and backend    |
| `npm run test:web`        | Tests frontend web application                                        |
| `npm run test:api`        | Tests backend API                                                     |
| `npm run test`            | Concurrently tests frontend and backend                               |
| `npm run build`           | Build the web production site to `./dist/`                            |
